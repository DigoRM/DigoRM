## ⚡ Executive Highlights & Top Projects

Here is a curated showcase of my main projects, representing full-stack development, machine learning, and enterprise RPA:

### 🤖 Enterprise Automation & RPA (Node.js & Python)
*   **[Automate Reports](https://github.com/DigoRM/AutomateReports)**: A enterprise-level monorepo (Node.js/React) designed for multi-tenant franchise reporting. Automates the parsing of massive Excel files and dynamically compiles customized, boardroom-ready PowerPoint (PPTX) and PDF presentations, saving franchise operations hundreds of hours of manual compilation.
*   **[Automate Emails](https://github.com/DigoRM/AutomateEmails)**: A robust transactional mailing system built with Node.js and SQLite. It handles high-volume queue messaging, CRM notifications, and segmented massive email dispatches with mock API keys and safe generic databases.

### 📊 AI & Financial Data Analytics (Python & Streamlit)
*   **[CRM & Customer Intelligence Suite](https://github.com/DigoRM/CRM-Intelligence-Suite)**: An interactive analytics application combining unsupervised learning (**K-Means Clustering** plotted on a stunning 3D Plotly canvas) and supervised learning (**Random Forest Churn Predictor**) to identify at-risk bank accounts. Features a batch CSV/Excel processing tool.
*   **[PerformaCX](https://github.com/DigoRM/CX_performance_movi)**: A customized productivity analyzer for Customer Experience (CX) operations. It processes raw helpdesk data (Movidesk) and implements a dynamic, mathematically-weighted productivity score algorithm to rank agents fairly based on tickets, speed, and time utilization.

### 🌐 Scalable Web Applications (React, Tailwind & Django)
*   **[Quick PDF Compiler](https://github.com/DigoRM/PDF-Transformer)**: A premium client-side web product built with React, Vite, and Tailwind CSS. It features a sleek glassmorphic drag-and-drop workspace where users can reorder, rotate, compress, and compile multiple images into a single high-quality PDF. *Includes built-in monetization slots for Google AdSense.*
*   **[FoodzOnline (Multi-Vendor Marketplace)](https://github.com/DigoRM/multivendor-appfood)**: A full-scale UberEats/iFood clone built on Django. Features distinct merchant/customer onboarding, geolocation coordinates mapping for proximity searches, and nested menu builders.
*   **[Django Portfolio CMS](https://github.com/DigoRM/portifolio_carreer)**: A fully normalized, database-driven Content Management System to dynamically manage, update, and showcase projects, courses, releases, and skill profiles.
